## Motor: Codex CLI

Usar para: implementaciones de múltiples archivos, migraciones, refactors grandes,
generación de tests exhaustivos, tareas que toman >10 minutos.

Modo: autónomo con gate de revisión al final.
Prerrequisito: especificación clara y detallada antes de delegar.
Salida: PR o diff revisable antes de merge.

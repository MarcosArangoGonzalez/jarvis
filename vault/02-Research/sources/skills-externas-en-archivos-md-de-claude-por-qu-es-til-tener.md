Vamos a explicarte
qué son y para qué es útil tener archivos .md con Skills
de
Claude . La
inteligencia artificial
de Anthropic tiene este sistema de
habilidades que pueden ser como los
GPTs
o
Gems
de
ChatGPT
o
Gemini , pero funcionan de una manera diferente.
Claude va a buscar entre sus habilidades cada vez que le escribas un prompt, y por eso es útil tener algunas externas en vez de dentro del sistema de Skills para poder usarlas únicamente cuando tú lo quieres. Vamos a explicártelo todo para que lo entiendas, y de paso te diremos la mejor manera de hacerlas sin tener que usar ningún programa de edición de texto.
Por qué es útil tener skills externas
Cuando decides crear una
Skills de Claude , esta se guarda en la memoria interna de la IA, lo que es normal. Pero la particularidad de este sistema respecto a otros similares de otros chatbots de IA es que cada vez que escribas un prompt,
Claude primero mirará los nombres y descripciones de todas las Skills
que tengas creadas por si puede utilizar alguna, y luego, si entiende que lo que has pedido corresponde con lo que hacen, entonces las lanza automáticamente.
Esto puede hacer que
no tengas el control total de cuándo se usa una skill
que has creado, y que se lancen sin que tú lo quieras. Y si vas creando muchas de ellas para diferentes tareas, entonces puede ser un problema.
La manera de evitar esto es
crear nombres y descripciones muy precisos
para las skills, de forma que Claude pueda entender para qué son específicamente y no lanzarlas en otros contextos. Si son descripciones y nombres genéricos, entonces podría usarlas sin que lo quieras con acciones similares que le pidas.
Pero el método para tener un mejor control y tener
una Skill que solo se use cuando tú quieras
explícitamente es el de usar una externa. Vamos, tener
archivos .md con estas Skills , y adjuntarla solo cuando quieras usarlas explícitamente.

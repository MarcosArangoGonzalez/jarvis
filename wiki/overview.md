# Jarvis Overview

JarvisOS is a local-first agentic environment for Marcos Arango.

## Layers

- Raw capture: unprocessed data in `raw/`.
- Wiki memory: curated knowledge in `wiki/`.
- Skills and MCP: automations and external adapters in `tools/`.
- Templates: strict Markdown note structures in `templates/`.
- Voice: local STT/TTS via whisper.cpp and Piper.

## Initial Focus

- Software engineering with senior-level architecture and security discipline.
- TFG research around Software Architecture and Security Specs.
- BJJ/Pesas tracking with metrics and recovery context.
- Personal automation and SaaS research-to-MVP.

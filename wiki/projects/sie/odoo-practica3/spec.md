# Especificacion

## Contexto

La practica pide personalizar Odoo con un modulo para la gestion de productos de una empresa que vende audiolibros.

## Requisitos Funcionales

- Crear los modelos necesarios para gestionar audiolibros.
- Registrar para cada audiolibro:
  - autores;
  - titulo;
  - productor;
  - duracion;
  - portada;
  - genero;
  - formato;
  - ISBN;
  - precio.
- Definir reglas de seguridad para manejar el catalogo.
- Crear relaciones entre los modelos del modulo.
- Definir menus y vistas para visualizar comodamente el catalogo.
- Permitir visualizar, introducir, modificar y borrar audiolibros.

## Requisitos de Entrega

- Practica por parejas y obligatoria.
- Fecha limite: 15 de mayo de 2026.
- Entrega en el campus virtual UDC.
- Un unico archivo comprimido.
- Nombre del archivo: `practica3_<apellido1>_<apellido2>_<nombre>.<extension>`.
- Incluir codigo del modulo desarrollado y comentado adecuadamente.
- Se puede anadir memoria PDF de no mas de 2 paginas.
